# Gaia
Lifesteal Utils' Common Networking Service for Real-time Data Syncronization and Collective Analytics

## Modules
* `collectivum` - uses the power of many. receptive endpoints for LSU clients to send in-game insights, like baltop records.
* `curiositas` - client-specific views. these are restricted endpoints that are only invoked by the client itself.
* `imperium` - feature-specific endpoints. provides a rest-like api for LSU-specific features.
